# Sosyal Medya Yönetim Aracı

Tüm sosyal medya hesaplarının merkezi yönetimi, içerik planlama, zamanlama ve analitik paneli

## Teknolojiler

FastAPI, Python, React, Next.js, PostgreSQL, Redis, Docker

## Başlangıç

```bash
# Kurulum
chmod +x setup.sh && ./setup.sh

# Başlatma
chmod +x start.sh && ./start.sh
```

## Proje Yapısı

Detaylı yapı için `DOSYA_YAPISI.md` dosyasına bakın.

## Emare Ekosistemi

Bu proje **Emare ekosisteminin** bir parçasıdır.

- 📁 Ortak çalışma dosyaları: `EMARE_ORTAK_CALISMA/`
- 🎨 Web tasarım: `web_dizayn/`
- 📝 Proje hafızası: `sosyal medya yönetim aracı_HAFIZA.md`
