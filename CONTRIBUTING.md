# Sosyal Medya Yönetim Aracı — Katkı Rehberi

## Nasıl Katkıda Bulunulur

1. Bu repo'yu fork edin
2. Feature branch oluşturun: `git checkout -b feat/yeni-ozellik`
3. Kodunuzu yazın (EMARE_ANAYASA.md kurallarına uygun)
4. Test ekleyin
5. Commit yapın: `git commit -m "feat: yeni özellik eklendi"`
6. Push: `git push origin feat/yeni-ozellik`
7. Pull Request açın

## Commit Mesaj Kuralları

```
feat: yeni özellik
fix: hata düzeltme
docs: dokümantasyon
refactor: yeniden yapılandırma
test: test ekleme/düzeltme
chore: build/config
```

## Kodlama Standartları

- EMARE_ANAYASA.md'deki kurallara uy
- Type hint kullan (Python)
- Docstring yaz
- Test ekle
